# matery

这是`Eclipse Color Themes`中的一个黑色质感`Ecllipse`主题。

## 使用方式

> **注意**：在`Eclipse`中需要安装`Eclipse Color Themes`的插件。

下载该项目并解压，在`Eclipse`中导入`theme-48979.xml`文件，然后选中`matery`应用即可。
